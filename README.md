# モダンなウェブサイト

HTML、CSS、JavaScriptを使用して作成されたシンプルで美しいウェブサイトです。

## ファイル構成

- `index.html` - メインのHTMLファイル（ページの構造）
- `styles.css` - スタイルシート（デザインとレイアウト）
- `script.js` - JavaScriptファイル（インタラクティブな機能）
- `README.md` - このファイル

## 使い方

1. すべてのファイルを同じフォルダに保存します
2. `index.html`をブラウザで開きます
3. または、ローカルサーバーを起動して表示します

### ローカルサーバーの起動方法

#### Python 3を使用する場合:
```bash
python3 -m http.server 8000
```

#### Node.jsを使用する場合:
```bash
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスします。

## カスタマイズ方法

### 色を変更する
`styles.css`の`:root`セクションで色を変更できます：

```css
:root {
    --primary-color: #6366f1;  /* メインカラー */
    --secondary-color: #8b5cf6;  /* サブカラー */
    /* ... */
}
```

### コンテンツを変更する
`index.html`を編集して、テキストやセクションを変更できます。

### 機能を追加する
`script.js`に新しい機能を追加できます。
